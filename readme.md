// Aprire un pacchetto di figurine

Comprare un pacchetto di figurine :
  - ?SE ho i soldi 
    * andare alla cartolibreria di fiducia :
      - prendere il portafoglio
      - uscire di casa
      - ?SE è lontana
        * usare un mezzo
      - :ALTRIMENTI
        * andare a piedi
  - :ALTRIMENTI
    * restare a casa
  - pagare la cartolibreria
  - tornare a casa

Aprire il pacchetto di figurine
  - recarsi in camereretta
    - ?SE porta chiusa
      * aprire porta
  - poggiare il pacchetto di figurine sulla scrivania
  - prendere un paio di forbici 
  - tagliare 2mm sotto il lato più corto
  - estrarre il contenuto dal pacchetto

Controllare il contenuto del pacchetto
  - recuperare l'album dalla libreria accanto la scrivania
  - 



Applicare nuove figurine sull' album




Scambiare le figurine doppie con gli amici di scuola
